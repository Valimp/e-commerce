# E-commerce SaaS

## Objectives
Create an e-shop SaaS easy to deployed.
This e-shop have to be environmentally friendly.

## Tech Stack
### Frontend
NextJs - Tailwindcss

### Backend
NodeJs - (MongoDb &/or Postgresql) - express

## Get started

```
git clone https://github.com/Valimp/e-commerce.git [DIRNAME]

cd [DIRNAME]

npm install

npm run dev
```
